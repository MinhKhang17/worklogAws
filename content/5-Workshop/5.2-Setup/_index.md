ℹ️ **Security Best Practices**
The AWS resources and code deployed through the course of this workshop should be removed immediately after you have completed the workshop. We’ve provided instructions in this guide to help you do that (see the *Cleanup* section). If you choose to use this workshop’s content as the basis for your own production applications, be sure to apply the security best practices for each service involved. You can find the security best practices documented in the “Security” section of each service’s Developer Guide, often in a subsection titled “Security Best Practices”. See [https://docs.aws.amazon.com/security/](https://docs.aws.amazon.com/security/) for convenient links to the appropriate documentation.

## Important Notes

1. When using the NICE DCV client, you may receive a message saying 'No license available'. A separate license for your Reality Capture Virtual Desktop is not required. The NICE DCV server will automatically detect that it is running on an Amazon EC2 instance and periodically connects to verify the license. This message is safe to disregard.

2. A notification about the NVIDIA Control Panel not being found may be displayed. This message is safe to disregard and will not affect performance or functionality.
